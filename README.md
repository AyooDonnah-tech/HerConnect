HerConnect - Alumni Portal

A centralized, sustainable web-based platform designed to connect, engage, and empower over 1,000+ alumni across 15+ cohorts from the Educating the Children community. This platform transforms fragmented, cohort-based WhatsApp groups into a unified digital sisterhood.

📌 Problem Statement
The alumni community currently lacks a central digital hub, causing cohorts to function independently with limited cross-interaction. Communication regarding jobs, scholarships, and tech events remains one-directional. This project provides a structured space for alumni to collaborate, organize discussions, showcase achievements, and foster leadership structures like "Sisterhood Fridays."

 🚀 Core MVP Features (1-Week Sprint)
Alumni Directory: A searchable database categorized by cohort year, major, and industry to bridge the gap between 15+ cohorts.
User Profiles: Individual spaces for alumni to display achievements, current job titles, and contact information.
Opportunities & Bulletin Board:A dedicated section for sharing jobs, scholarships, internships, and mentorship requests.
Events Tracker: A centralized calendar to organize and coordinate community initiatives and "Sisterhood Fridays."
